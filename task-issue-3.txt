## Your Task

Implement the Hub event bus for pub/sub communication.

### Functions to Implement

- `hub_emit(EventType, TargetID, data)` — emit an event to all subscribers
- `hub_subscribe(EventType, handler, userdata)` — subscribe to an event type
- `hub_unsubscribe(EventType, handler)` — unsubscribe

Components emit events when their state machines transition. Other components (like bar) subscribe to these events.

### Code Location

Add to existing `wm-hub.c` and `wm-hub.h`.

### Acceptance Criteria

- [ ] Multiple subscribers can receive the same event
